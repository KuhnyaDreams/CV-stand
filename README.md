
# CV-стенд для исследования атак и защит на CV-модели

Стенд для экспериментов с атаками и защитой CV-систем. Основан на YOLOv26 (Ultralytics), разделён на ядро (API) и обёртку (скрипты для атак). Ядро запускается в Docker-контейнере.

---

### 1. Сборка (один раз)
```bash
docker-compose build
```
Первая сборка может занять 20–30 минут (скачивание модели, зависимостей).

### 2. Запуск
```bash
docker-compose up -d
```
- Ядро доступно: http://localhost:8000
- Jupyter (опционально): http://localhost:8888

### 3. Запуск экспериментов

Через IDE запускаете конкретный файл

### 4. Остановка
```bash
docker-compose down
```
